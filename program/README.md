# SPQR program

## Pipeline

### Generate arguments

Depending on the entrypoint:
```
make shield-args
make transfer-args
make unshield-args
```

### Produce execution trace

```
make artifacts
```

### Generate proof

```
make prove
```

### Split proof into chunks

```
make calldata
```

### Register proof on Sepolia

```
make register-fact
```
